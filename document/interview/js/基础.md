## null == undefined ?

* **true** 
* null 表示没有对象，即此处不应该有值
* undefined 表示缺少值，就是此处应该有值，但是还没有定义

## JSON转对象再转回来

* JSON.stringify() 方法将一个 JavaScript 值（对象或者数组）转换为一个 JSON 字符串
* JSON.parse() 方法用来解析 JSON 字符串，构造由字符串描述的 JavaScript 值或者对象
